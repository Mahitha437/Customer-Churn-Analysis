# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data from a telecom company to identify factors that influence customer retention and churn.

## Dataset
Dataset: WA_Fn-UseC_-Telco-Customer-Churn.csv
Number of records: 7032
Number of features: 21

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code
- Git & GitHub

## Visualizations
1. Customer Churn Distribution
2. Gender vs Churn
3. Contract Type vs Churn
4. Internet Service vs Churn
5. Monthly Charges Distribution

## Key Findings
- Most customers did not churn.
- Customers with month-to-month contracts showed higher churn.
- Internet service type influences churn behavior.
- Monthly charges vary significantly among customers.

## Conclusion
Customer churn can be reduced by improving retention strategies for high-risk customers, especially those with short-term contracts and higher monthly charges.
